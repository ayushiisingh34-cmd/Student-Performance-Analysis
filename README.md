# 📊 Student Performance Analysis

## Objective

This project analyzes student data to understand how study hours, attendance, previous grades, parental support, and extracurricular activities affect final grades.

## Dataset Features

* StudentID
* Name
* Gender
* AttendanceRate
* StudyHoursPerWeek
* PreviousGrade
* ExtracurricularActivities
* ParentalSupport
* FinalGrade

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Steps

1. Loaded and cleaned the data.
2. Explored the dataset.
3. Created charts and graphs.
4. Studied relationships between different factors.
5. Found useful insights from the data.

## Key Findings

* Students who study more usually get better grades.
* Better attendance is linked to better performance.
* Previous grades have a strong effect on final grades.
* Parental support helps improve student performance.
